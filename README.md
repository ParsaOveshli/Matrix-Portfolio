## ✨ درباره پروژه

**Matrix-Portfolio** یک پرتفولیوی جذاب و متفاوت است که با الهام از فضای دیجیتالی Matrix طراحی شده.
این پروژه علاوه بر طراحی گرافیکی چشم‌نواز، دارای قابلیت‌های امنیتی و تعاملی زیر است:

### 🔐 ویژگی‌ها

* 🚫 **غیرفعال بودن Inspect Element**
* 🛡️ **جلوگیری از Copy / Cut / Paste** برای حفاظت از کد
* 🔒 **غیر فعال بودن راست-کلیک**
* ⚡ **افکت ماتریکسی و UI مدرن**
* 📱 **کاملاً ریسپانسیو روی موبایل و دسکتاپ**
